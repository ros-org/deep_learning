### **光伏板异物分类项目 **



该项目目标是识别光伏板上的异物，区分有异物和没有异物两种情况。

使用分类模型： RegNet 。

模型基础权重： RegNet_Y_16GF_Weights.IMAGENET1K_SWAG_E2E_V1
